# My Asteroid
This is a simple reproduction of Asteroids built in Python and guided by [boot.dev](https://boot.dev).
There is no score-keeping and the goal is to survive as long as possible.
## Controls
| Action | Control |
| ---    |  ---    |
| Move Forward | W |
| Move Backward | S |
| Rotate Counter-Clockwise | A |
| Rotate Clockwise | D |
| Shoot | Space |
